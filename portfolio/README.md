# Portfolio

A modern, professional portfolio website for a final-year Diploma IT student aspiring to become a Data Analyst / Power BI Developer.

## Features
- Fully responsive, mobile-first design
- Clean, modern dark theme with blue/cyan accents
- Sticky, animated navbar with hamburger menu
- Dynamic projects page (loads from JSON)
- Honest, student-focused content
- No frameworks or libraries (HTML, CSS, JS only)

## Structure
```
portfolio/
├── index.html        (Home)
├── projects.html     (Projects)
├── about.html
├── contact.html
├── css/
│   ├── base.css
│   ├── navbar.css
│   ├── animations.css
│   └── pages.css
├── js/
│   ├── main.js
│   └── projects.js
├── data/
│   └── projects.json
├── images/
│   └── (project screenshots / profile image)
└── README.md
```

## Deployment
- 100% static, deployable to GitHub Pages or Netlify

## Customization
- Replace images in `images/` with your own
- Edit `data/projects.json` to update projects
- Update content in HTML files as needed
